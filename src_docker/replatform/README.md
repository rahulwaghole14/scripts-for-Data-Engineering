# hexa-data-replatform-analysis

Author: Thomas Kliem


This project is to download data from Adobe Analytics and upload to BigQuery for a Power BI to show how the replatform is performing compared to the previous site.

Tested using python 3.10.2

Requires secrets\private.key and secrets\secret_values.json from Adobe and secrets\hexa-data-report-etl-prod-f89b703bc217.json from Google.

Run create_credentials.py then main.py
