# Naviga Data to BigQuery Workflow

This repository contains Python scripts for loading data into a Google BigQuery table.

## Files

- `naviga_data_to_bq.py`: A Python script that fectches data from naviga api and uploads its contents to a specified BigQuery table.
- `naviga-workflow.dockerfile`: Related Dockerfile to create a containerized environment for running the Python script.
